# speckit-autopilot: coverage-report

> **Deprecated** — use `/audit` instead, which includes coverage reporting plus AI review per feature.

Generate a static coverage report (structural gaps + file counts per feature).

## Usage

```bash
speckit-autopilot coverage-report --root .
```

For a full quality audit including AI review, use:
```bash
speckit-autopilot audit --root .
```
